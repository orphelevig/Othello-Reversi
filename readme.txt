Bienvenue !

Vous trouverez ici le premier vrai projet de mes études supérieures en informatique :
L'implémentation du jeu de société "Othello" ou "Reversi" en java !

Il a été réalisé en groupe avec deux de mes camarades de classe (bonjour à vous si vous passez par là ;) )

Vous trouverez dans le dossier ici le code Reversi.java pouvant être exécuté à condition d'avoir
le dossier class avec (Start.class et SimpleInput.class indispensable).
Il faudra que le Reversi.java se trouve dans un répertoire 'src' au même niveau que le répertoire 'class' pour l'exécuter.
Ces fichiers ont été là uniquement pour le 1er semestre de mes études afin de nous cacher la partie
orientée objet de java (choix de nos enseignants).

Le code et l'affichage dans la console peuvent parraître brutaux mais pour un premier vrai projet 
en java (ça faisait 3 mois qu'on en faisait) on est plûtot fiers du résultat !

En espérant que vous puissiez jouer une ou deux parties avant de vous en aller !
